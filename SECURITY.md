# Security Policy

## Supported Versions
Only the **latest released version** is supported with security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest release | ✅        |
| Older releases | ❌        |

## Reporting a Vulnerability
If you discover a security vulnerability, please help us disclose it responsibly.

- **Do not** open a public GitHub issue.
- Report it privately via:
  - **Email:** nmk8rf2kwt@privaterelay.appleid.com
  - **GitHub:** Use the repository's **Private vulnerability reporting** / **Security Advisories** feature (if enabled).

We will review reports as soon as possible and get back to you.
